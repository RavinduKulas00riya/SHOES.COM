A modern and user-friendly e-commerce web application for browsing and purchasing shoes online.

## Features
- User authentication (Login & Sign up)
- Browse shoes by size and preferences
- Product search with filtering, sorting & pagination
- Editable shopping cart
- Secure checkout process
- Responsive and modern UI

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

## How to Run Locally
1. Install XAMPP / WAMP
2. Place the project in the `htdocs` folder
3. Import the database (if available)
4. Update database credentials
5. Run via `http://localhost/shoes.com`
